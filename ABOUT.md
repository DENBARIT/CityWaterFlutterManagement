Longer project summary (for README or project page):

AquaConnect is a unified platform for municipal water utilities. It provides an administrative web dashboard, a REST API backend with authentication and billing automation, and a Flutter mobile client for field agents and customers. The system uses PostgreSQL with Prisma for persistence, supports SMTP for OTP and notifications, and is designed for local deployment and cloud hosting.
